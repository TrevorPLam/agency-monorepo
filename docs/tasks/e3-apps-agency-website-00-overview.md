# E3 Apps Agency Website


## Purpose
Build the agency's public marketing website as a Next.js App Router application with modern SEO, performance, and content management capabilities. This is the primary inbound channel for new client acquisition.


## Dependencies
- `30-ui-theme` - for design tokens
- `31-ui-icons` - for iconography
- `32-ui-design-system` - for components
- `80-analytics` - for tracking
- `40-seo` - for metadata
- `51-data-cms` and `e8-apps-studio` when optional content workflows are activated


## Success Criteria

This agency website validates:
- ✅ Marketing-ready Next.js App Router implementation
- ✅ SEO-first architecture with dynamic metadata
- ✅ Performance monitoring integration
- ✅ Analytics tracking implementation
- ✅ CMS integration ready (optional)
- ✅ Lead generation capabilities
- ✅ Accessibility compliance
